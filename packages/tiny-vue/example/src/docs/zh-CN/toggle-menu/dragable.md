## ToggleMenu 收缩菜单

<nova-uxlink widget-name="ToggleMenu"></nova-uxlink>

以树形方式显示、适用于版型定制的一种组件，菜单可收缩，收缩后只显示图标，点击图标展示子节点菜单。

</div>

### 拖拽节点

可通过 `draggable` 属性设置是否开启拖拽节点的功能，默认值为 `false`。

可通过 `ellipsis` 属性设置是否开启文本内容超长是省略显示，默认值为 `false`
