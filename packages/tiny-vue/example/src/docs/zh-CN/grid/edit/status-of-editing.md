## Grid 表格

<nova-uxlink widget-name="Grid"></nova-uxlink>

表格组件，提供了非常强大数据表格功能，在 Grid 可以展示数据列表，可以对数据列表进行选择、编辑等。
</div>

### 开启编辑状态

 表格属性设置 `edit-config` 开启编辑模式，然后在该属性对象内设置 `showStatus` 开启或关闭单元格更新状态（单元格左上角倒三角形更新标识)，默认值为 `true` 开启状态。
