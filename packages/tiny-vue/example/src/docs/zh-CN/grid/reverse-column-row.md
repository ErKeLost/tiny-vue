## Grid 表格

<nova-uxlink widget-name="Grid"></nova-uxlink>

表格组件，提供了非常强大数据表格功能，在 Grid 可以展示数据列表，可以对数据列表进行选择、编辑等。
</div>

需要在 `grid` 标签上配 `:columns="tableColumn"` `:show-header="false"`。点击“反转表格”按钮可查看效果，现阶段只能执行一次，执行多次会有问题。具体参考下面示例。
