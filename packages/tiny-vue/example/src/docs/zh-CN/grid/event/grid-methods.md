## Grid 表格

<nova-uxlink widget-name="Grid"></nova-uxlink>

表格组件，提供了非常强大数据表格功能，在 Grid 可以展示数据列表，可以对数据列表进行选择、编辑等。
</div>

### 获取表格行方法
::: tip 获取行的方法
  `getCurrentRow` 获取当前行
  `getRowIndex` 获取当前行号
  `getRadioRow` Radio 单选选中行
  `getRowById` 通过 rowId 获取当前行
  `getRowNode` 通过 tr 元素获取行信息
:::
