## Grid 表格

<nova-uxlink widget-name="Grid"></nova-uxlink>

表格组件，提供了非常强大数据表格功能，在 Grid 可以展示数据列表，可以对数据列表进行选择、编辑等。

### 基本用法

通过属性 `column-width` 设置所有列宽度，默认值为均匀分配。
