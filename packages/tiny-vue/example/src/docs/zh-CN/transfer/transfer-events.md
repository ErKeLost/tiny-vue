### 穿梭框事件

主要有 `change`、`left-check-change`、`right-check-change` 三个事件。

:::tip 事件说明
change：右侧列表元素变化时触发
left-check-change：左侧列表元素被用户选中 / 取消选中时触发
right-check-change：右侧列表元素被用户选中 / 取消选中时触发
:::
