## Select 选择器

<nova-uxlink widget-name="Dropdown"></nova-uxlink>

Select 选择器是一种通过点击弹出下拉列表展示数据并进行选择的 UI 组件。

</div>

### 选项可清除

通过 `clearable` 属性可将选择器清空，鼠标悬停于选择器上时，会出现清除图标。需要注意的是，`clearable` 属性仅适用于单选。
