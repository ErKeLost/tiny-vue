## DatePicker 日期选择器

<nova-uxlink widget-name="DatePicker"></nova-uxlink>

用于设置/选择日期，包括年月/年月日/年月日时分/年月日时分秒日期格式。
</div>

### 清除输入

选择日期后，鼠标悬停在输入框时，默认会显示清除图标，单击图标可以清除输入框内容。设置 `clearable` 属性为 false 后，则不显示清除图标，不可清除。通过 `clear-icon` 属性可以自定义清除图标。
