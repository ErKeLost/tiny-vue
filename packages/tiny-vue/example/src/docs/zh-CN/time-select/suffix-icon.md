## TimeSelect 时间选择器

<nova-uxlink widget-name="DropTimes"></nova-uxlink>

时间选值框是下拉框的特例，按时分格式（不支持自定义格式）展示下拉列表以供用户选择，并能扩展下拉框部分参数。
</div>

### 后置图标

通过 `suffix-icon` 属性设置日期输入框后置图标，从 `@opentiny/vue-icon` 中导入一个图标并进行初始化后传给 `suffix-icon`。
