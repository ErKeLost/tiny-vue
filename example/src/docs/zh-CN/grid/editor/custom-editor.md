<nova-uxlink widget-name="Grid"></nova-uxlink>

表格组件，提供了非常强大数据表格功能，在 Grid 可以展示数据列表，可以对数据列表进行选择、编辑等。

### Grid-编辑器-自定义编辑器

`grid` 标签上配置 `edit-config`。
在 `grid-column` 列配置 `editor` 对象， `component` 渲染自定义编辑组件。
自定义编辑默认编辑框为小框，可以配合样式修改编辑框。
