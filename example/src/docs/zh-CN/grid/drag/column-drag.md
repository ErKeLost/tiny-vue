## Grid 表格

<nova-uxlink widget-name="Grid"></nova-uxlink>

表格组件，提供了非常强大数据表格功能，在 Grid 可以展示数据列表，可以对数据列表进行选择、编辑等。

### 列拖拽

通过设置 `drop-config` 的 `column` 属性控制列拖拽，默认为开启列拖拽，列拖拽事件有 `@column-drop-start`、`column-drop-move`、`column-drop-end`。
