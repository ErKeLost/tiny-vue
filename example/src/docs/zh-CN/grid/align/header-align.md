## Grid 表格

<nova-uxlink widget-name="Grid"></nova-uxlink>

表格组件，提供了非常强大数据表格功能，在 Grid 可以展示数据列表，可以对数据列表进行选择、编辑等。
</div>

### 表头左对齐

`header-align` 设置表头对齐方式可选值为 `left` `center` `right`，默认值为 `left`。

:::tip
表格上配置 `header-align` 会被列配置的 `header-align` 覆盖
:::
