## Grid 表格

<nova-uxlink widget-name="Grid"></nova-uxlink>

表格组件，提供了非常强大数据表格功能，在 Grid 可以展示数据列表，可以对数据列表进行选择、编辑等。
</div>

### 滚动到指定位置

:::tip 方法说明
`scrollTo(scrollLeft, scrollTop)`：滚动到对应的位置
`scrollToRow(row)`：滚动到对应的行
`scrollToColumn(column)`：手滚动到对应的列
:::
