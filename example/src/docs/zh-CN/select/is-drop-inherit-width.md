### 下拉列表跟随输入框宽度

通过 `is-drop-inherit-width` 属性设置下拉列表是否跟随输入框宽度。默认为 `false` 表示不跟随，内容可撑开下拉列表宽度，设置为 `true` 则跟随宽度。
