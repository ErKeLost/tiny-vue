## DialogBox 对话框

<nova-uxlink widget-name="DialogBox"></nova-uxlink>

模态对话框，在浮层中显示，引导用户进行相关操作。
</div>

### 弹出与关闭事件

`open` ：对话框打开时触发，`opened` ：对话框打开动画结束时触发，`close` ：对话框关闭时触发，`closed` ：对话框关闭动画结束时触发。
