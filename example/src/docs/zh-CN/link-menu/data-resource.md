## LinkMenu 收藏夹菜单

<nova-uxlink widget-name="LinkMenu"></nova-uxlink>

LinkMenu 收藏夹形式的菜单，支持可配置菜单项。

</div>

### 数据源

通过 `data` 属性设置菜单的数据源，同时在数据对象中可通过 `disabled` 设置该节点是否可被勾选。
