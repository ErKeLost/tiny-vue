## LinkMenu 收藏夹菜单

<nova-uxlink widget-name="LinkMenu"></nova-uxlink>

LinkMenu 收藏夹形式的菜单，支持可配置菜单项。
</div>

### 可收藏栏目数

通过 `max-item` 属性指定可勾选并收藏的栏目数。指定 `default-expand-all` 为 false 时，打开菜单弹窗时所有节点为折叠状态。
