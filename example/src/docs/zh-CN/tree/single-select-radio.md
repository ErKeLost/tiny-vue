## Tree 树形控件

<nova-uxlink widget-name="Tree"></nova-uxlink>

可进行展示有父子层级的数据，支持选择，异步加载等功能。但不推荐用它来展示菜单，展示菜单推荐使用树菜单。
</div>

### 单选

通过 `show-radio` 属性设置树节点的单选特性，默认为 `false`，与 `show-checkbox` 属性互斥，不能同时设置为 `true`。
